# 📋 Инструкция для преподавателя

Как создать репозиторий для нового ученика курса.

## 🔧 Предварительная настройка (один раз)

### 1. Создать template репозиторий

1. Создать новый репозиторий `swot-template` в своём GitHub
2. Загрузить туда содержимое архива
3. Settings → ☑️ Template repository

### 2. Получить API ключ Anthropic

1. Зарегистрироваться на https://console.anthropic.com
2. Создать API ключ
3. Сохранить — он будет использоваться для всех учеников

---

## 👤 Для каждого нового ученика

### Шаг 1: Создать репозиторий

1. Перейти в template репозиторий
2. Нажать **Use this template** → **Create a new repository**
3. Заполнить:
   - Owner: можно твой аккаунт или ученика
   - Repository name: `swot-[имя-ученика]` (например: `swot-ivan`)
   - ☑️ Private (рекомендуется)
4. Нажать **Create repository**

### Шаг 2: Добавить API ключ

1. Открыть созданный репозиторий
2. **Settings** → **Secrets and variables** → **Actions**
3. Нажать **New repository secret**
4. Name: `ANTHROPIC_API_KEY`
5. Secret: вставить API ключ
6. **Add secret**

### Шаг 3: Включить GitHub Pages

1. **Settings** → **Pages**
2. Source: **GitHub Actions**
3. Сохранить

### Шаг 4: Дать доступ ученику

**Вариант A: Ученик работает в твоём репо**
1. **Settings** → **Collaborators**
2. **Add people** → ввести GitHub username ученика
3. Выбрать роль **Write**

**Вариант B: Репо на аккаунте ученика**
- Создать репо сразу на аккаунте ученика (Owner = ученик)
- API ключ всё равно добавляешь ты

### Шаг 5: Отправить ученику

Отправить ученику:
```
Твой репозиторий для SWOT-анализа готов!

🔗 Репозиторий: https://github.com/[owner]/[repo-name]
🌐 Визуализация: https://[owner].github.io/[repo-name]/

Как начать:
1. Склонируй репозиторий в VS Code
2. Открой context.md — опиши свой бизнес
3. Открой analysis/company.md — опиши текущую ситуацию
4. Сохрани (Ctrl+S), закоммить и запуш (Ctrl+Shift+G)
5. Через 3 минуты обнови (Pull) и смотри результат

Подробная инструкция в README.md
```

## 🔄 Типичный workflow ученика

```
Ученик в VS Code:
     │
     ▼
┌─────────────────────┐
│ Редактирует         │
│ context.md          │
│ analysis/company.md │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ Commit + Push       │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ GitHub Actions      │
│ (2-3 минуты)        │
│ - SWOT анализ       │
│ - Коммит результата │
│ - Деплой на Pages   │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ Ученик делает Pull  │
│ Смотрит результат:  │
│ - outputs/*.md      │
│ - GitHub Pages      │
└─────────────────────┘
```

## ❓ Решение проблем

### Workflow не запускается
- Проверить что изменились файлы `context.md` или `analysis/*.md`
- Проверить что пуш в ветку `main`

### Ошибка "ANTHROPIC_API_KEY not set"
- Проверить название секрета (точно `ANTHROPIC_API_KEY`)
- Проверить что ключ валидный

### GitHub Pages не работает
- Settings → Pages → Source должен быть "GitHub Actions"
- Первый деплой может занять до 5 минут

### Ученик не видит результат
- Напомнить сделать Pull после пуша
- Подождать 3-5 минут
- Проверить Actions на ошибки

## 📊 Мониторинг

### Проверить статус анализа
1. Открыть репозиторий ученика
2. Вкладка **Actions**
3. Последний запуск — зелёная галочка = успех

### Посмотреть результат
- **outputs/swot_latest.md** — текстовый отчёт
- **https://[owner].github.io/[repo]/** — визуализация
